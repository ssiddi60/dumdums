Savings Help

The Savings page allows you to add or subtract from the current total. The current total is always diplayed so you know exactly how much money is allocated in savings.

  • Add to Current Total: To add to the current total, type an amount in the textfield and click the "+" button. The current total will reflect the update.
  • Subtract from Current Total: To subtract from the total, type an amount in the textfield and click the "-" button. The current total will refelct the update. If the current total is 0, the "-" button will be disabled. If the amount you want to subtract exceeds the current total, the current total will be 0.
